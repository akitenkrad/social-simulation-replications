# {{NAME}} 再現実装

論文の再現実装．Cargo workspace + uv workspace の 2 プロジェクト構成．

## 構成

- `simulation/` — Rust シミュレーション．クレート `{{NAME}}-simulation`，バイナリ `{{NAME}}`．
- `tools/` — Python 製可視化・分析ツール．パッケージ `{{NAME}}-tools`（src レイアウト，`tools/src/{{NAME}}_tools/`）．
- `results/` — シミュレーション結果の出力先（gitignore 対象）．

## ビルド & 実行

全コマンドは workspace ルート（本ディレクトリ）から実行する．

```bash
# Rust シミュレーション
cargo build --release
cargo run --release -- run --help
cargo run --release -- sweep --help

# Python ツール
uv sync
uv run {{NAME}}-tools --help
uv run {{NAME}}-tools visualize --help
uv run {{NAME}}-tools visualize-sweep --help
uv run {{NAME}}-tools show-experiment-settings --results-dir results/latest
```

`visualize-sweep` はパラメータ依存図に加えて，各パラメータ組み合わせを格子状に並べた
進行アニメーション (`sweep_grid_animation.gif`) を生成する．これは sweep を
`--snapshot-interval N` (N>0) 付きで実行した場合のみ生成される（スナップショットが
無ければ警告のみ出してスキップ）．

## 出力構造

各実行はタイムスタンプ付きサブディレクトリに保存される．`results/latest` は最新の実行へのシンボリックリンク．

```
results/
├── latest -> 20260101_120000
├── 20260101_120000/
│   ├── metrics.csv
│   ├── config.json          ← 実行時の設定値（show-experiment-settings で表示可能）
│   └── snapshots/
└── ...
```
